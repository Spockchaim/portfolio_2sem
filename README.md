## 📋 Sobre o Projeto

O objetivo deste projeto é apresentar as competências técnicas, projetos, formação acadêmica e participação em eventos de Pedro Chaim em um layout moderno, futurista e elegante. 

Diferente de portfólios estáticos tradicionais, esta aplicação foi estruturada como um **SPA dinâmico client-side**, onde todas as informações de projetos, habilidades e certificados são gerenciadas através de um arquivo de dados estruturado em JavaScript (`script.js`). Isso garante carregamento instantâneo, separação de dados e facilidade de manutenção.

### ✨ Principais Recursos Visuais & UX

- **Aparência Premium Dark:** Fundo obsidian profundo (`#030712`) com efeitos de profundidade gerados por blobs de gradiente brilhantes e flutuantes.
- **Glassmorphism:** Cards e barra de navegação com efeitos de vidro translúcido (`backdrop-filter: blur`).
- **Animações de Entrada:** Barras de progresso de habilidades que carregam com efeitos suaves apenas quando entram na área de visualização do usuário (`IntersectionObserver`).
- **Navegação Inteligente:** Cabeçalho flutuante que encolhe no scroll e destaca automaticamente a seção ativa conforme o usuário navega na página.
- **Responsividade Total:** Adaptado perfeitamente para dispositivos móveis, tablets e telas widescreen, com menu mobile otimizado (hambúrguer).
- **Sem Dependência de Servidor:** Funciona 100% no navegador (client-side), facilitando o deploy instantâneo em plataformas gratuitas como GitHub Pages, Vercel ou Netlify.

## 🛠️ Stack Tecnológica

- **Estruturação:** HTML5 Semântico para SEO avançado.
- **Estilização:** Vanilla CSS3 avançado (Flexbox, Grid, Variáveis CSS, Backdrop Blur, Keyframes).
- **Lógica & Renderização:** JavaScript (ES6+) moderno para manipulação de DOM e animações por IntersectionObserver.
- **Ícones:** Bootstrap Icons e Devicon (para logotipos de tecnologias).
- **Tipografia:** Plus Jakarta Sans & Outfit (Google Fonts).

## 🚀 Como Executar Localmente

Como o projeto é um SPA client-side sem necessidade de backend, executá-lo é extremamente simples:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Spockchaim/portfolio_2sem.git
   cd portfolio_2sem
   ```

2. **Abra no Navegador:**
   - Basta abrir o arquivo `index.html` diretamente em qualquer navegador moderno.
   - *Alternativa sugerida (VS Code):* Utilize a extensão **Live Server** para rodar com live-reload local.
